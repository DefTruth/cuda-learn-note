# Sigmoid

## 0x00 说明

包含以下内容：

- [X] sigmoid_f32_kernel
- [X] sigmoid_f32x4_kernel(float4向量化版本)
- [X] PyTorch bindings


## 测试

```bash
# 只测试Ada架构 不指定默认编译所有架构 耗时较长
export TORCH_CUDA_ARCH_LIST=Ada 
python3 sigmoid.py
```
